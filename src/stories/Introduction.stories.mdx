## How to Use This Storybook  
Let's dive in to how to use this.  
<br /><br />

### The Color system/TypeScale system  

Both the color and typescale within the primitives folder serves as a display of current settings for design system within this storybook. Anyone may change this setting to fit their needs.
For each component, there are a choices of color or typescale types that you can choose from. A color/typescale type here means something like a brand color or background color, a color that is
defined by its usage, such as "Extra Large Title" or "Brand Color". The reason it is set up this way is to stay coherent with the design system. In order to change a color in this color system, read the instructions 
provided below. <br />
1. Go to Primitives/Colors/"Name of the color type that you want to change" or TypeScale/"Name of the typescale type that you want to change"
<br />
2. Change color in "color"/"size" field in the panel 
<br />
3. Click the "Click to save current customized ...." button to save 
<br />
4. Now all components using that color will have the new color/typescale 
<br />
### The Grid System

The Grid System also serves as a display of what can be done with Bootstrap css, not really meant to be used, unless it is exactly what you're looking for.
Please refer to https://getbootstrap.com/docs/4.0/layout/overview/ for more information on how to use Bootstrap.
